GW-Router (Catalyst / AS65001) に BGP を設定する
未完了

CatalystにBGPプロセスを起動し、Si-Rをネイバーとして登録します。

# AS65001でBGPを起動
GW-Router# router bgp 65001
# Si-R(10.0.0.2)をAS65002のネイバーとして登録
GW-Router(config-router)# neighbor 10.0.0.2 remote-as 65002
# 自分のネットワークをBGPで広告
GW-Router(config-router)# network 192.168.10.0 mask 255.255.255.0
3
Router-A (Si-R / AS65002) に BGP を設定する
未完了

Si-RはCisco形式に近いコマンドでBGPを設定します。

# AS65002でBGPを起動
Router-A# router bgp 65002
# Catalyst(10.0.0.1)をAS65001のネイバーとして登録
Router-A(config-router)# neighbor 10.0.0.1 remote-as 65001
# 自分のネットワークを広告
Router-A(config-router)# network 192.168.1.0 mask 255.255.255.0
💡 Si-RのBGPコマンドはCiscoとほぼ同じ形式です

📚 学習ポイント

  • BGPはAS間の経路制御プロトコル（iBGP=AS内、eBGP=AS間）
  • TCP 179番ポートを使用（信頼性のある接続）
  • メトリックではなくポリシー（AS_PATH, LOCAL_PREF等）でパス選択
  • eBGP AD値=20、iBGP AD値=200
  • AS番号 64512〜65534 = プライベートAS（組織内使用）
  • neighbor コマンドで対向IPと相手AS番号を指定
  • 異ベンダー間（Cisco ↔ Si-R）でも同じコマンド体系で設定可能
