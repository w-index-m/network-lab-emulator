📚 学習ポイント

  • OSPFはリンクステート型 — ネットワーク全体のトポロジーを把握
  • DR/BDR選出: priority大 → Router ID大 の順
  • priority=0 はDR/BDR選出から除外（DROther固定）
  • ネイバー状態: Down → Init → 2Way → ExStart → Exchange → Loading → Full
  • コスト = 参照帯域幅(100M) ÷ インタフェース帯域幅
  • AD値 = 110（RIPの120より小さく優先）
  • wildcard mask = サブネットマスクのビット反転（0.0.0.255 = /24）

⚡ DR/BDR 選出ルール

優先度条件結果
1位priority 最大DR に選出
2位priority が同じ → Router ID 最大DR に選出
除外priority = 0DR/BDR 非対象
注意一度選出されたDRは変わらない非プリエンプティブ
